Place assets here. Next.js serves /public at the site root (e.g. /images/...).

Hero slider (MainHero):
- Name files banner-1.jpg, banner-2.webp, etc. (prefix "banner", any image extension).
- The /api/banners route lists matching files for the homepage slider.

Inner pages (InnerBanner):
- Pass backgroundImage="/images/your-file.jpg" from each page.
- Demo assets: inner-about.svg, inner-services.svg, inner-utilities.svg, inner-industrial.svg,
  inner-projects.svg, inner-certifications.svg, inner-stakeholders.svg, inner-contact.svg

Other assets (logos, etc.) can use any other filename pattern.
